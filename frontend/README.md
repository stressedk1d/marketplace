# VogueWay — фронтенд

Next.js (App Router), TypeScript, Tailwind.

## Запуск

```bash
npm install
npm run dev
```

Переменная окружения: `NEXT_PUBLIC_API_URL` (по умолчанию `http://localhost:8000`) — URL бэкенда.

Полное описание проекта, API и миграций: [**README.md в корне репозитория**](../README.md).
